    Introducción a la Asignatura: Lenguajes de Marcas

    ¿Qué son los Lenguajes de Marcas?

    Los lenguajes de marcas (en inglés, markup languages) son sistemas de codificación utilizados para estructurar, identificar y presentar información dentro de documentos digitales. Son esenciales para organizar el contenido en la web y permiten que la información sea legible tanto para las máquinas como para las personas.

    Características principales

    • Estructuración: Organizan el contenido en etiquetas y atributos.
    • Compatibilidad: Son legibles tanto por humanos como por máquinas.
    • Flexibilidad: Permiten la personalización de la presentación del contenido, lo que facilita el estilizado y manipulación de los documentos.

    Tipos de Lenguajes de Marcas

    1. HTML (HyperText Markup Language): Es el lenguaje de marcas más utilizado para crear páginas web. Define la estructura de las páginas web mediante etiquetas como <html>, <head>, <body>, entre otras.

    2. XHTML (Extensible HyperText Markup Language): Es una versión más estricta y formateada de HTML que sigue las reglas de XML. Puede utilizarse cuando se quiere garantizar que el código sea más limpio y estandarizado.

    3. XML (eXtensible Markup Language): A diferencia de HTML, XML no define etiquetas fijas. Permite a los usuarios crear etiquetas personalizadas para almacenar datos de manera estructurada y coherente.

    4. CSS (Cascading Style Sheets): Aunque no es un lenguaje de marcas, CSS es esencial para aplicar estilos visuales a las páginas HTML, mejorando la presentación del contenido estructurado.

    5. Python: Aunque Python no es un lenguaje de marcas, su uso en la web como lenguaje de programación backend es clave para generar sitios web dinámicos e interactivos. Usamos Python para la manipulación de datos y la creación de servidores web.

    Objetivos de la Asignatura

    Conocimiento fundamental de los Lenguajes de Marcas

    El principal objetivo es proporcionar una base sólida en HTML, XHTML, XML, y CSS, permitiendo que el alumno comprenda cómo estructurar documentos web y cómo estos lenguajes influyen en el diseño y la presentación de una página web.

    Capacitación en el uso de herramientas modernas de creación de documentos web

    Se trabajarán herramientas modernas como Markdown, CSS para estilizar páginas web y Python para crear aplicaciones interactivas que utilicen lenguajes de marcas.

    Aplicación práctica en proyectos reales

    Además de las explicaciones teóricas, los alumnos desarrollarán proyectos prácticos donde podrán aplicar los conocimientos adquiridos para crear páginas web estáticas y dinámicas utilizando los lenguajes de marcas y otras tecnologías web.

    Contenido del Curso

    1. Introducción a HTML y su Sintaxis

    Se analizará la sintaxis de HTML, su estructura básica, etiquetas y atributos.

    2. Características de XHTML y su Compatibilidad con HTML

    Se hará una comparativa entre HTML y XHTML, destacando las diferencias más importantes.

    3. Uso de XML para Almacenar Datos

    Se introducirá el uso de XML para crear archivos estructurados y su uso en aplicaciones de gestión de datos.

    4. Creación de Contenido en CSS

    Se mostrará cómo utilizar CSS para crear el diseño y el estilo de una página web.

    5. Introducción a Python para el Backend de la Web

    Se explorará el uso de Python en el backend, especialmente en el desarrollo de aplicaciones web con frameworks como Flask o Django.

    Conclusión

    Los lenguajes de marcas son esenciales para el uso efectivo de la web, ya que permiten estructurar y presentar el contenido de manera clara y accesible. Esta asignatura proporcionará las habilidades necesarias para comenzar a trabajar con estos lenguajes y aplicarlos en proyectos web reales.
